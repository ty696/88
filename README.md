# 帝辇洗护系列品牌官网

这是一个专业的精品洗护品牌官网，采用现代简约设计，完全响应式，支持手机和桌面端访问。

## 项目结构

```
dinian/
├── index.html          # 首页
├── products.html       # 产品范围页面
├── news.html          # 新闻动态页面
├── about.html         # 关于我们页面
├── contact.html       # 联系我们页面
├── css/
│   ├── styles.css     # 主样式文件
│   └── navigation.css # 导航栏样式
├── js/
│   └── main.js        # 主要JavaScript功能
└── images/
    ├── products/      # 产品图片文件夹
    ├── news/          # 新闻图片文件夹
    └── about/         # 关于我们图片文件夹
```

## 功能特性

### 导航栏
- 固定在页面顶部
- 左侧显示品牌Logo（长条型图片）
- 导航菜单：首页、产品范围、新闻动态、关于我们、联系我们
- 移动端响应式菜单（汉堡菜单）
- 当前页面高亮显示

### 首页功能
1. **轮播图**
   - 自动播放（5秒切换）
   - 手动切换按钮
   - 指示器导航
   - 鼠标悬停暂停

2. **产品范围展示**
   - 8个产品系列展示
   - 卡片式布局
   - 悬停动画效果
   - 响应式网格布局

3. **公司信息**
   - Logo展示
   - 公司地址
   - 联系电话

### 产品范围页面
- 详细展示各个产品系列
- 支持不同尺寸的图片（自动适配）

### 新闻动态页面
- 新闻列表展示
- 图文并茂
- 日期显示

### 关于我们页面
- 品牌介绍
- 公司使命
- 核心价值

### 联系我们页面
- 联系信息展示
- 在线留言表单
- 表单验证

## 设计特点

- **现代简约**：采用简洁大气的设计风格
- **响应式设计**：完美适配手机、平板、桌面端
- **优雅配色**：金色主题色，体现高端品牌形象
- **流畅动画**：平滑的过渡效果和交互体验
- **图片适配**：自动适配不同尺寸比例的图片

## 使用说明

1. 将所有图片按照 `images/README.md` 中的说明放置到对应文件夹
2. 在浏览器中打开 `index.html` 即可查看网站
3. 建议使用本地服务器运行（如使用VS Code的Live Server插件）

## 技术栈

- HTML5
- CSS3（Flexbox、Grid）
- JavaScript（原生JS，无依赖）

## 浏览器支持

- Chrome（推荐）
- Firefox
- Safari
- Edge

## 图片要求

- 格式：JPG、PNG
- Logo：长条型图片，建议尺寸约 200×50px 或等比例
- 产品图片：支持各种尺寸（长条型、9:16、正方形等）
- 如果图片不存在，网站会自动显示占位符
